Membership

We are asking for $30 for a yearlong membership, so we can host more workshops and events, help fund members so they can attend conferences such as SPAC and RLDC and create projects. Membership benefits include:

  • Access to our Electrical Lab
    • Soldering stations
    • Oscilloscopes
    • Function generators
    • Power supplies; common components and tools
  • Lab Computers and Printers
    • Common engineering and CS software is installed; lab managers can add new software on request
  • Microwave, Fridges, Freezer
    • Project Lab Space and Storage
  • Students may request an IEEE project box to store hardware for class or personal projects

Come to our lab room, ICS 225, if you would like to become a member.
